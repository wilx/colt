MIF Doclet

Last modified: Thu Jul 29 12:37:11 PDT 1999

Javadoc Home Page

The MIF Doclet is an extension to Javadoc that generates API documentation in MIF (Maker Interchange Format) from declarations and doc comments in source code written in the JavaTM programming language. These MIF documents can then be opened in Adobe FrameMaker (or fmbatch) and optionally converted to PDF, PS, Microsoft RTF, Microsoft Word or WordPerfect.

Support Note - The MIF Doclet is experimental code developed for internal use at Sun and released to the developer community for use as-is. It is not a supported product. Use at your own risk. While we have worked to make it useful internally, bugs might not be fixed and features might not be added. It might not be updated, or might be re-released with a different software or GUI design possibly incompatible with the current design.

Download

Download MIF Doclet. It requires version 1.2.2 (or later) of the JavaTM 2 SDK, Standard Edition.
